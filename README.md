# 👟 Adidas Sales Analytics Dashboard | Power BI

## 📊 Project summary 

This Power BI dashboard provides a comprehensive analysis of Adidas sales performance across products, retailers, regions, states, and sales channels. The dashboard helps identify revenue trends, profitable regions, top-performing products, and customer purchasing behavior for data-driven business decisions.

---

## 🎯 Business Objective

The main objective of this dashboard is to:

* Analyze overall sales performance.
* Identify top-selling products.
* Compare retailer contributions.
* Evaluate regional profitability.
* Track monthly sales trends.
* Understand sales channel performance.
* Discover high-performing states.

# 📂 Adidas US Sales Dataset Overview

This dashboard is built using the **Adidas US Sales Dataset**, which contains sales transactions from multiple retailers across different regions, states, products, and sales channels across the United States.

---

# 📊 Dataset Summary

| Attribute | Description |
|------------|-------------|
| Dataset Name | Adidas US Sales Dataset |
| Total Records | 9,648 |
| Total Columns | 13 Business Fields |
| Industry | Retail & Sportswear |
| Analysis Type | Sales Analytics & Business Intelligence |
| Tool Used | Power BI |

---

# 📋 Dataset Columns Explanation

| Column Name | Description |
|-------------|-------------|
| Retailer | Name of the retailer selling Adidas products |
| Retailer ID | Unique identifier assigned to each retailer |
| Invoice Date | Date when the transaction occurred |
| Region | Sales region (West, Northeast, Southeast, South, Midwest) |
| State | State where the sale was made |
| City | City where the sale occurred |
| Product | Adidas product category |
| Price per Unit | Selling price of one product unit |
| Units Sold | Number of units sold |
| Total Sales | Revenue generated from sales |
| Operating Profit | Profit earned after sales |
| Operating Margin | Percentage of profit earned |
| Sales Method | Sales channel (In-store, Outlet, Online) |
